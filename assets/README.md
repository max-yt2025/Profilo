## Image Specifications:

### Profile Images:
- Main profile: 400x400 pixels (square)
- High-quality profile: 500x500 pixels or larger
- Format: JPG or PNG
- Keep file size under 100KB for faster loading

### Social Media Images:
- Open Graph: 1200x630 pixels (1.91:1 ratio)
- Twitter Card: 1200x600 pixels (2:1 ratio)
- Format: JPG or PNG
- Keep file size under 200KB

### Project Images:
- Thumbnail images: 400x300 pixels (4:3 ratio)
- Format: JPG or PNG
- Keep file size under 100KB each

## How to Create Placeholder Images:

1. **Use placeholder services** (for development):
   - https://placeholder.com/
   - https://via.placeholder.com/
   - https://placehold.co/

2. **Create custom images**:
   - Use image editing software (Photoshop, GIMP, Canva)
   - Maintain consistent style and branding
   - Optimize for web (compress images)

3. **Take screenshots** (for projects):
   - Capture screenshots of your actual projects
   - Crop and resize appropriately
   - Add text overlays if needed

## Image Optimization:

- Compress images before uploading
- Use modern formats (WebP) where supported
- Maintain good quality while keeping file sizes small
- Use descriptive filenames
- Add alt text in HTML for accessibility
