# Favicon Files

This directory should contain the following favicon files for the website:

## Required Files:

1. **favicon.ico** - Classic favicon for older browsers (32x32 or 16x16)
2. **favicon-16x16.png** - PNG favicon for modern browsers (16x16)
3. **favicon-32x32.png** - PNG favicon for modern browsers (32x32)
4. **apple-touch-icon.png** - Apple touch icon for iOS devices (180x180)
5. **android-chrome-192x192.png** - Android Chrome icon (192x192)
6. **android-chrome-512x512.png** - Android Chrome icon (512x512)

## Optional Files:

7. **favicon-48x48.png** - Additional PNG size (48x48)
8. **mstile-150x150.png** - Microsoft Edge tile (150x150)

## How to Create Favicons:

1. Design your favicon (simple is better for small sizes)
2. Use online tools like:
   - https://favicon.io/
   - https://realfavicongenerator.net/
   - https://www.favicon-generator.org/
3. Upload your design and download all the generated files
4. Place the files in this directory
